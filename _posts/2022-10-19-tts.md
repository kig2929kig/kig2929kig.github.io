# TTS(Text to Speech)  

텍스트 음성 변환 

```python
import sys, os

try :
    import pyttsx3
except ImportError :
    print("\npyttsx3 모듈이 설치되지 않았습니다. 다음 명령어를 입력하여 모듈을 설치해 주세요.")
    print("pip install pyttsx3\n")
    #os.system('pip install pyttsx3')
    sys.exit()

tts = pyttsx3.init()

print("문자를 음성으로 변환합니다. 종료하려면 QUIT를 입력하세요.")
while True :
    text = input('>>>')

    if text.upper() == 'QUIT':
        print('프로그램 종료')
        sys.exit()
    tts.say(text)
    tts.runAndWait()
```
