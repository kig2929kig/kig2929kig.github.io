## 1. CSV 파일 불러오기

+ PassengerId : 승객ID
+ Survived: 생존 여부 `0(yes)` 또는 `1(no)`
+ PClass : 티켓 클래스 - '1', '2', '3'
+ Name : 승객의 이름
+ Sex : 성별
+ Age : 나이
+ SibSp : 형제자매 또는 배우자 수
+ Parch : 부모 또는 자녀 수
+ Ticket : 티켓 번호
+ Fare : 운임 == 요금
+ Cabin : 객실 번호
+ Embarked : 승선 포트(port)

import pandas as pd
